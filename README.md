# 🏥 BioNexus - Centro de Bienestar Tecnológico

BioNexus es una plataforma web integral diseñada para un centro de salud de vanguardia que combina biología humana con tecnología de precisión. Este proyecto fue desarrollado con un enfoque en la experiencia de usuario (UX), diseño responsivo y validación de datos en el lado del cliente.

## 🚀 Características Principales

- **Diseño Moderno & Responsive:** Interfaz adaptada a dispositivos móviles y escritorio utilizando arquitecturas de Flexbox y CSS Grid.
- **Validación Avanzada de Formularios:** Uso de **Expresiones Regulares (Regex)** en JavaScript para garantizar la integridad de los datos (nombres, teléfonos y correos).
- **Interactividad Dinámica:** Manipulación del DOM para mostrar confirmaciones de contacto en tiempo real sin recargar la página.
- **Arquitectura de Estilos Escala:** Implementación de **Variables CSS** para una gestión de colores y temas eficiente y consistente.
- **Navegación Fluida:** Incluye funciones globales como el botón "Volver arriba" para mejorar la usabilidad.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica para SEO y accesibilidad.
- **CSS3:** Diseño personalizado con variables, transiciones y sistema de rejillas.
- **JavaScript (Vanilla):** Lógica de validación, manejo de eventos y almacenamiento temporal de datos.
